How it works

The script downloads various IETF data sources daily, runs a diff against yesterday's data, and displays the changes as an HTML page. The data sources are the following:

  • ID Tracker database dump
  • Internet-Draft abstracts
  • IETF-Announce mailing list archive index page
  • IPR disclosure Atom feed
  • Liaison statements list
  • IESG minutes
  • IAB minutes
  • IAOC minutes
  • IETF trust minutes
  • RFC editor queue XML dump

Atom/RSS feeds

There's also an Atom and RSS feeds available. Currently, it only contains links to the daily pages, not the actual content.
